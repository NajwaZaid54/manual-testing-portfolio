# 🧪 Manual Testing Portfolio – SauceDemo Login Testing

## 📌 Project Overview

This project demonstrates manual testing performed on the SauceDemo login module. The testing process includes validating login functionality, input validation, error handling, and user interface behavior.

The project was created as part of a software testing learning portfolio to strengthen understanding of QA processes, test case creation, and bug identification.
